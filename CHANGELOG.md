# Changelog

All notable changes to TypeFace will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-23

### Added
- Initial release of TypeFace
- Font browser with search and filtering
- Live preview with custom text input
- Side-by-side font comparison (2-4 fonts)
- Seven scene templates: Title/Heading, Body Text, Logo, Poster, Social Media Post, Business Card, App UI
- Typography controls: size, weight, letter spacing, line height, color, alignment
- Favorites system for saving fonts and font combinations
- Font details view with metadata and character set preview
- High-resolution PNG export functionality
- Dark mode support
- Haptic feedback throughout the app
- Universal app support (iPhone + iPad)
- Full accessibility support
- Performance optimizations for smooth scrolling with hundreds of fonts
- SwiftData persistence for favorites and collections
- Complete CoreText integration for font metadata

### Technical
- Swift 5.9+
- SwiftUI-based interface
- MVVM architecture
- iOS 17.0+ minimum deployment
- Zero external dependencies
- Optimized for performance and battery life
